# Application bonjour avec nodejs

- nodejs
- express

## Structure du projet

Gestion_etudiants/

- node -v

## Installation

### 1. Cloner le depot

cd Gestion_etudiants

### 3. Installation des dependances
npm install

### 4. Lancer le projet
node server.js
